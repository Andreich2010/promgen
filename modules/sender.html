config(key)[source]¶

Plugin specific configuration

This wraps our PROMGEN settings so that a plugin author does not need to be concerned with how the configuration files are handled but only about the specific key.

send(data)[source]¶

Send out an alert

This handles looping through the alerts from Alert Manager and checks to see if there are any notification senders configured for the combination of project/service and sender type.

See tests/examples/alertmanager.json for an example payload
